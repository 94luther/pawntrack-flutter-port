# PawnTrack Flutter Port

This is a new Flutter web/mobile port of PawnTrack. It was created in its own folder so the existing PawnTrack HTML, React, backend, spreadsheet files, and zip artifacts stay untouched.

## Structure

- `app/` - Flutter app
- `server/` - Firestore-primary backend with Google Sheets sync
- `database/` - Firestore collection notes
- `docs/` - setup notes

Start with `docs/SETUP.md`.
